0. 복습
행 인덱스/ 열 이름 설정
- 행 인덱스 변경: df.index = 새로운 행 인덱스 배열 형태로 넣어준다.
- 열 이름 변경: df.columns = 새로운 열 인덱스 배열(list) 형태로 넣어준다.
- 일부 행 인덱스 변경: df.rename(index = {기존 인덱스: 새 인덱스, ...})
- 일부 열 인덱스 변경: df.rename(columns = {기존 열 이름: 새로운 열 이름, ...})

행/열 추가
- 열 추가: df['추가할 열 이름'] = 데이터
- 행 추가: df.loc['추가할 행 이름'] = 데이터

행/열 삭제
- df.drop(axis = 0): 행 삭제
- df.drop(axis = 1): 열 삭제

행/열의 위치 변경
- df.transpose()
- df.T

데이터 프레임 정렬
- df.sort_index() 


판다스 파일 입출력
: 판다스 패키지에서는 파일 입출력할 수 있는 내장 메서드들이 있다.

1) 파일 입출력
- 파일 입력(input): 기존의 파일 내용을 읽어들이는 것
- 파일 출력(output): 기존의 파일에 새로운 내용을 추가하거나 새로운 파일을 생성하는 것

2) 판다스의 파일 입출력 메서드(csv, excel)
- df.to_csv("파일명"): 판다스 객체를 csv로 저장
- df.to_excel("파일명"): 판다스 객체를 엑셀로 저장
- pd.read_csv("파일명"): 해당 파일 csv 파일을 데이터 프레임으로 읽어오기
- pd.read_excel("파일명"): 해당 엑셀 파일을 데이터 프레임으로 읽어오기

3) 데이터 내용 확인 메서드
- df.head() : 데이터 프레임의 일부 앞부분 내용 출력(처음 5개 행)
- df.tail(): 마지막 부분 내용 출력(마지막 5개 행)
- df.shape: 데이터 프레임의 크기(행의 개수, 열의 개수)를 확인 => 튜플 형태
- df.info(): 데이터 프레임에 관한 기본 정보를 화면에 출력
 	=> 클래스 타입, 행 인덱스의 구성, 열 이름의 종류와 개수, 각 열의 자료형과 개수, ...
- df.dtypes : 각 열의 자료형 확인
- df.describe() : 숫자 데이터를 갖는 열에 대한 주요 통계정보를 요약하여 출력
   => 평균, 표준편차, 최대값, 최소값, 중간값 등

4) 데이터 개수 확인
 (1) 열의 데이터 개수
	- df.count() : 각 열이 가지고 있는 데이터 개수를 series 형태로 반환
	- df.value_counts() : 각 열의 고유값의 종류와 개수를 확인, series 형태로 반환
 (2) 특정 행의 데이터 개수 확인
 	- df.loc['행이름'].count()
	- df.iloc[행의 idx].count()

 (3) 행의 개수
	- len(df)


5) 판다스 통계 함수 적용
- df.mean() : 각 열의 평균값
- df.median() : 각 열의 중간값
- df.max() : 각 열의 최대값
- df.std() : 각 열의 표준편차
- 특정 열에 통계함수 적용할 수도 있다.
	ex) df["열이름"].mean()
	ex) df[['열1', '열2']].max()


데이터 시각화 (data visualization)
: 데이터를 분석한 결과를 사용자가 쉽게 이해할 수 있도록 표현하여 전달하는 것

matplotlib
: 다양한 데이터들을 많은 방법으로 도식화할 수 있는 파이썬 라이브러리
[공식문서]https://matplotlib.org


설치 및 사용 방법
1) 설치
vs code 터미널
pip install matplotlib

2) 사용
import matplotlib
import matplotlib.pyplot as plt


3) 기본 설정
- plt.figure: 그래프 영역 설정
	=> figure(figsize = (가로길이, 세로길이))
- plt.show(): 그래프 출력

4) plot(): 꺾은 선형 그래프
	plt.plot(x, y): x축에서 사용할 데이터와 y축에서 사용할 데이터를 각각 리스트로 만든 뒤 
	인수로 전달 

5) 선 그래프 꾸미기 옵션
- marker: 마커 모양('o', '+', '*', '.', ...)
- markerfacecolor: 마커 배경색
- markersize: 마커 크기
- color: 선의 색깔
- linewidth: 선의 두께
- label: 라벨 지정



6) bar() : 막대 그래프
: 범주가 있는 데이터의 값을 직사각형의 막대로 표현하는 방법
어떤 데이터의 많고 적음을 나타낼 때 주로 사용
