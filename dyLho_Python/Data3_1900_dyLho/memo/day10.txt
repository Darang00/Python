0.복습

머신러닝
- 기계(컴퓨터 알고리즘) 스스로 데이터를 학습하여 서로 다른 변수간의 관계를 찾아가는 과정
- 예측, 분류, 군집 알고리즘 등으로 분류
- 머신러능은 크게 지도학습과 비지도 학습으로 분류
  지도학습 : 정답 데이터를 다른 데이터와 함께 알고리즘에 입력하는 방식(회귀분석, 분류)
  비지도학습 : 정답 데이터 없이 컴퓨터 알고리즘 스스로 데이터로부터 숨은 패턴을 찾아내는 방식

회귀분석
- 가격, 매출, 주가 등 연속적인 값을 갖는 값을 예측하는데 주로 사용
- 단순회귀분석, 다항회귀분석, 다중회귀분석



1. 분류 알고리즘
- 예측하려는 대상의 속성(독립 변수)을 입력받고, 목표 변수(종속 변수)가 갖고 있는 카테고리 값 중에서
  어느 한 값으로 분류하여 예측하는 알고리즘
- 분류 알고리즘은 고객 분류, 질병 진단, 스펨 메일 필터링 등 목표변수가 카테고리 값을 갖는 경우에 사용
- KNN, SVM, Decision Tree 등 다양한 알고리즘 존재
