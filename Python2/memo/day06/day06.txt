AI : 인공지능
학습하고 추론하고, 지각능력 같은 사람의 지능을 인공적으로 만드는 기술
머신러닝과 딥러닝은 인공지능을 구현하는 방법

머신러닝
인공지능의 하위 범주로서 경험(학습)을 기반으로 결과를 예측하는 기술
-> 머신러닝은 컴퓨터를 학습 시키고 학습한 내용을 기반으로 판단, 예측하여 인간이 내리는 의사결정을 대신한다.

참고 키워드
	Feature(특징) / Attribute(속성)
	머신러닝은 어떤 학습을 하더라도 결국 데이터를 분류하거나 값을 예측하는
	것이 목적이다.
	이런 예측이나 분류를 하기 위한 데이터의 특징 Feature라고 부른다.
	-> 칼럼

	Model(모형)	
	현실 세계에 있는 현상이나 데이터를 특정 목적에 맞게 수학적으로 표현한 것
	과거의 데이터를 통해 학습하여 모델을 만들고, 미래의 사건을 예측하거나
	분류하는데 사용된다.

	예시)
	- 사람
	1. 짱구는 소고기를 먹고 '맛있다', 피망을 먹고 '맛없다'라고 생각
	2. 경험을 통해 피망 또는 피망처럼 생긴것은 '맛없다'라는 판단력이 생김
	3. 짱구에게 돼지고기와 파프리카를 준다면 짱구는 먹어보지도 않고 돼지고기를
	선택할 것이다.

	- 컴퓨터
	1. 음식에 대한 짱구의 과거 반응을 분석한다.
	2. 학습을 통해 어떤 음식을 싫어하는지 특징을 찾아낸다.
	(패턴을 찾아내는 것이며, 이 패턴을 수식으로 만들어낸다.)
	3. 학습이 끝나면 이를 통해 짱구가 싫어할만한 음식을 가려낼 수 있다.
	(여기서 짱구가 싫어할 것이다 라고 판단할 때 사용한 수식을 모델이라고 한다.)

모델을 만드는 과정을 모델링이라 하며, 주어진 데이터에서 최적의 모델을 찾아야한다.

머신러닝 프로세스(진행 순서)
1. 문제 정의
	해결하려는 문제를 정의하고, 모델이 달성해야하는 목표를 설정한다.
2. 데이터 수집
	모델이 학습할 데이터를 수집하는 단계
3. 데이터 전처리
	수집한 데이터를 파악 및 결측치/중복값 등 잘못된 데이터를 처리한다.
4. 모델 선택
	문제 해결에 적합한 모델을 선택한다.
5. 모델 학습
	준비된 데이터로 선택한 모델을 학습 시키는 단계
	이 과정을 통해 모델은 데이터의 패턴을 학습한다.
	모델 학습에 사용되는 데이터를 훈련 데이터라고 한다.
6. 모델 평가
	학습이 끝난 모델로 예측/분류하여 결과를 평가하는 단계
	평가에 사용되는 데이터를 검증 데이터라고 한다.
7. 모델 조정 및 최적화
	모델의 성능을 개선하기 위해 조정
8. 모델 활용
	검증이 끝난 모델을 확정하고 실제 문제 해결에 적용하는 단계

머신러닝 학습의 종류
1. 지도학습
	정답이 있는 데이터로 학습을 시킨다.(문제 + 정답을 알려줌)

	- 분류 : 카테고리 분류
	주어진 데이터를 정해진 정답(카테고리)에 따라 분류하는 것
	두 가지로 분류하면 이진 분류, 세 가지 이상으로 분류하면 다중 분류

	- 회귀분석 : 수치 예측
	데이터들의 특징를 기준으로 연속되는 값들을 예측한다.
	변수들 간의 상관관계를 분석하여 독립변수가 종속변수에 미치는 영향을 파악한다.
	이를 통해 독립변수(원인)의 변화에 따른 종속변수(결과)의 변화를 예측한다.		

2. 비지도학습
	정답이 없는 데이터로 학습을 시킨다.
	- 비지도 학습의 종류
	1. 군집 분석
	

3. 강화학습
	데이터가 없을 수 있으며 데이터가 있어도 정답이 없다.
	자신이 한 행동에 대한 보상을 준다.

회귀 분석
	- 회귀 분석의 종류
	1. 선형 회귀 분석(Linear Regression)
	선형 모델을 분석하는 기법	
	- 단순 선형 회귀
	- 다항 회귀
	- 다중 선형 회귀
	- 그 외 여러가지....

	2. 비선형 회귀 분석(NonLinear Regression)
	비선형 모델을 분석하는 기법
	- 이차 회귀 
	- 로그 회귀
	- 기타 등등

예시 )
	- 선형 회귀
	사과 1개 : 1000원
	사과 2개 : 2000원

	즉, 사과가 2배가되면 가격도 정확히 2배 -> 사과와 가격이라는 두 변수의 관계가
	선형적이다.

	- 비선형 회귀
	공부 시간과 시험 성적
