중복 데이터 처리

1) 중복 데이터 확인
- df.duplicated() : 전에 나온 행들과 비교해서 중복되는 행이 있다면 True, 없다면 False

2) 중복 데이터 제거
- df.drop_duplicates() : 중복되는 행을 제거하는 메소드, inplace = True
- subset = [열1, 열2,...] : 해당하는 열 기준으로 중복 데이터 삭제


범주형 데이터 처리

1) 구간 분할
: 연속성을 가지는 데이터를 일정한 구간(bin)으로 나누는 것
- pd.cut() : 데이터를 여러 구간으로 분할 하는 함수

2) 더미변수
: 카테고리를 나타내는 범주형 데이터는 분석 알고리즘에 바로 사용할 수 없는 경우가 있기 때문에
데이터를 더미 변수로 변환하여 사용한다
- 더미 변수는 True 또는 False로 표현된다
- pd.get_dummies() : 범주형 변수의 모든 고유값을 각각 새로운 더미 변수로 변환한다

시계열 데이터
- 인덱스가 날짜 혹은 시간인 데이터
- 1) Timestamp : 특정 시점을 표현 
  2) Period : 두 시점 사이의 일정한 기간 

- to_datetime() : 다른 자료형을 판다스 Timestamp객체로 변환
- to_period() : Timestamp객체(특정 시점)를 일정한 기간을 나타내는 Period 객체로 변환


시계열 데이터 만들기
- date_range() : 여러개의 Timestamp객체가 들어 있는 배열 형태의 시계열 데이터 생성
- period_range() : 여러개의 Period 객체가 들어 있는 시계열 데이터 생성



함수 매핑
- 시리즈, 데이터 프레임의 개별 원소를 특정 함수에 일대일 대응시키는 과정'
- 사용자 정의 함수, lamda 적용 가능
- 판다스 기본 함수로 처리하기 어려운 연산을 판다스 객체에 적용할 수 있다.


1. 시리즈 함수 매핑
1) 시리즈 원소에 함수 매핑
 - sr.apply(함수)
 - 인수로 전달되는 함수에 시리즈의 모든 원소를 하나씩 전달하고 리턴값을 반환받는다.
 - 같은 크기의 시리즈 객체로 반환된다.

2) 시리즈 객체에 함수 매핑
(1) df.apply(함수, axis = 0): 데이터 프레임의 각 열에 함수 매핑
(2) df.apply(함수, axis=1) : 데이터 프레임의 각 행에 함수 매핑
- 데이터 프레임의 모든 열이나 행을 분리하여 함수의 인수로 전달된다
- 각 열이나 행은 시리즈 형태로 반환 -> 반환된 시리즈들이 하나의 데이터프레임으로 통합



2. 데이터 프레임 함수 매핑
1) 데이터 프레임 원소에 함수 매핑
 - df.map(함수)
 - 인수로 전달되는 함수에 데이터 프레임의 원소를 하나씩 전달하고 리턴값을 반환받는다
 - 동일한 형태의 데이터 프레임이 반환된다.

2) 데이터프레임 객체에 함수 매핑
 - df.pipe(함수)
 - 함수가 반환하는 리턴값에 따라서 데이터 프레임, 시리즈, 스칼라로 값을 반환할 수 있다.
