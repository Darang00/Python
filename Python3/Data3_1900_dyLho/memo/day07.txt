0. 복습

1) 그룹 객체 생성
 - df.groupby(기준이 되는 열)
 - 특정 그룹 추출: 그룹객체.get_group(key)

2) 그룹객체 함수 매핑
 - 그룹객체.agg(매핑함수)
 - 모든 열에 여러 함수 매핑: 그룹객체.agg([함수1, 함수2, ...])
 - 각 열마다 다른 함수 매핑: 그룹객체.agg({"열1" : 함수, "열2", [함수2, 함수3, ...]})

3) 그룹 객체 필터링
 - 그룹객체.filter(조건식 함수): 조건이 참인 그룹들을 반환한다.


피벗 테이블
 - 엑셀에서 사용하는 피벗테이블과 비슷한 기능을 가진다.
 - 피벗 테이블: 데이터에서 필요한 특정 데이터만 뽑아 새롭게 테이블을 만드는 기증
 - 피벗 테이블의 구성요소
	pd.pivot_table(df, #피벗할 데이터프레임
                           index = '행에 위치할 열',
			   columns = '피벗테이블의 열에 들어갈 열',
			   values = '데이터로 사용할 열',
             		   aggfunc = '데이터 집계함수')
