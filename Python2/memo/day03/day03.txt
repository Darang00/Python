API
	API는 한 프로그램이 다른 프로그램의 기능이다 데이터를 사용할 수 있게 하는 방법이다.

Open API (= public API)
	각자 URL이 있음. 그 URL에 요청을 보내면 그 URL(API)가 응답을 함.
	응답할 때 HTML같이 필요없는거 빼고 데이터만 보내줌.
	- 외부에 공개된 api를 의미한다.
	- 요청을 보내면 응답에 데이터가 담겨서 온다.
	

API의 응답 데이터
	오늘날의 Open API 는 응답데이터를 담을 때 JSON 형식으로 데이터를 보내주는 경우가 많다.
	JSON은 csv, xml 같이 데이터를 표현하는 방식일 뿐이다.
	

ex) 
1번 학생은 김철수 나이는 22살
2번 학생은 김영희 나이는 24살

[CSV]
번호, 이름, 나이
1, 김철수, 22
2, 김영희, 24



JSON
	1. 데이터를 저장하거나 통신으로 전달할 때 사용하기 좋은 형식
	2. 데이터를 표현하는 형식이기 때문에 단순 텍스트이다.
	3. 사람이 읽고 쓰기 편하게 설계되었으며, 기계가 파싱하기도 쉽다.

JSON의 형식
	- JSON은 기본적으로 key와 value 한 쌍으로 이루어진다.
	파이썬의 dict와 매우 유사하여 JSON을 dict로 변환하여 다루는 경우가 많다.

	1. JSON 객체
	{
		"키1": 값1,
		"키2": 값2
	}
	JSON 객체는 중괄호 한 쌍으로 감싸지며, 키는 항상 "쌍따옴표"로 감싼다.


	2. JSON 배열
	JSON 객체를 담은 배열
	[{ }, { }, { }, .... ]
	JSON 배열은 대괄호로 감싸준다. (dict를 여러개 감싸고 있는 list라고 생각해도 괜찮음)
