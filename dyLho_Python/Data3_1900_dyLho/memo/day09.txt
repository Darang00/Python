머신러닝
- 기계(컴퓨터 알고리즘) 스스로 데이터를 학습하여 서로 다른 변수 간의 관계를 찾아가는 과정
- 해결하고자 하는 문제에 따라 예측, 분류, 군집 알고리즘 등으로 분류
- 예시
   1) 예측 : 주가, 환율 등 경제지표 예측
   2) 분류 : 은행에서 고객을 분류하여 대출을 승인하거나 거절
   3) 군집 : 비슷한 소비패턴을 가진 고객 유형을 군집으로 묶음
- 머신러닝은 다양한 영역에서 사용될 수 있고, 사용하는 알고리즘과 방법이 무수히 많다
- 머신러닝은 크게 지도학습과 비지도학습으로 분류된다

1) 지도학습 vs 비지도학습
지도학습 : 정답 데이터를 다른 데이터와 함께 알고리즘에 입력하는 방식
비지도학습 : 정답 데이터 없이 컴퓨터 알고리즘 스스로 데이터로부터 숨은 패턴을 찾아내는 방식

      			지도 학습            비지도학습
알고리즘(분석모형)  	회귀분석,            군집분석
      			분류

특징      1) 정답을 알고 있는 상태에서 학습    1) 정답이 없는 상태에서 서로
                  			비슷한 데이터를 찾아서 그룹화
      	 2) 모형 평가 방법이 다양한 편       2) 모형 평가 방법이 제한적


2) 머신러닝 프로세스
(1) 컴퓨터 알고리즘에 이해할 수 있는 형태로 데이터를 변환하는 작업
	- 분석 대상에 관해 수집한 관측값(데이터프레임 행)을 속성(데이터프레임 열)을 기준으로 정리
	=> 판다스를 이용해 데이터프레임으로 정리하는 과정

(2) 여러 속성(변수)간의 관계를 분석하여 결과를 예측하는 모형을 학습을 통해 찾는 과정
	- 훈련 데이터를 통해 모형을 학습시킨다.

(3) 학습을 마친 모형의 예측 능력을 평가하는 과정
	- 검증 데이터를 통해 모형의 예측 능력을 평가

(4) 최종 모형을 확정하여 문제 해결에 적용

데이터 정리 -> 데이터 문리(훈련/검증) -> 알고리즘 준비 -> 모형 학습(훈련 데이터)
-> 예측(검증 데이터) -> 모형 평가 -> 모형 활용



1. 회귀분석
- 머신러닝 알고리즘 중에서도 비교적 이해하기 쉽고, 활용성이 높은 알고리즘
- 가격, 매출, 주가 수량 등 연속적인 값을 갖는 데이터를 예측하는데 주로 활용
- 분석모형이 에측하고자 하는 변수: 종속 변수 or 예측 변수
- 예측을 위해 모형이 사용하는 속성: 독립 변수 or 설명 변수

1) 단순회귀분석
- 독립변수와 종속변수 두 변수 사에에 일대일 대응되는 확률적, 통계적 상관성을 찾는 알고리즘
- 일대일 대응관계 : 선형관계
   => 1차원 함수 : y = ax + b
- 즉, 단순회귀분석 알고리즘은 훈련 데이터를 통해 직선의 기울기(a)와 y절편(b)를 찾는 과정
- sklearn 라이브러리에서 LinearRegression() 함수로 회귀분석 모형 객체를 생성
