0.복습

머신러닝
- 기계(컴퓨터 알고리즘) 스스로 데이터를 학습하여 서로 다른 변수간의 관계를 찾아가는 과정
- 예측, 분류, 군집 알고리즘 등으로 분류
- 머신러능은 크게 지도학습과 비지도 학습으로 분류
  지도학습 : 정답 데이터를 다른 데이터와 함께 알고리즘에 입력하는 방식(회귀분석, 분류)
  비지도학습 : 정답 데이터 없이 컴퓨터 알고리즘 스스로 데이터로부터 숨은 패턴을 찾아내는 방식(군집)

회귀분석
- 가격, 매출, 주가 등 연속적인 값을 갖는 값을 예측하는데 주로 사용
- 단순회귀분석, 다항회귀분석, 다중회귀분석



1. 분류 알고리즘
- 예측하려는 대상의 속성(독립 변수)을 입력받고, 목표 변수(종속 변수)가 갖고 있는 카테고리 값 중에서
  어느 한 값으로 분류하여 예측하는 알고리즘
- 분류 알고리즘은 고객 분류, 질병 진단, 스펨 메일 필터링 등 목표변수가 카테고리 값을 갖는 경우에 사용
- KNN, SVM, Decision Tree 등 다양한 알고리즘 존재

KNN(K - Nearest - Neighbors)
- K개의 가까운 이웃의 목표 변수값으로 예측하는 알고리즘
- 관측값(행)과 기존 데이터 중에서 가장 속성이 비슷한 K개의 이웃 탐색
- 가까운 이웃들이 갖고 있는 목표 값 중 가장 많이 나온 값으로 분류하여 예측

분류 모형 평가 지표
(1) Confusion Matrix: 모형의 예측값과 실제 값을 각각 축으로 하는 매트릭스로 지표를 표현
- TP(True Positive): 긍정예측을 성공 즉, 승객이 생존했다고 예측했고 실제 생존함
- TN(True Negative): 부정예측을 성공 즉, 승객이 생존하지 못했다고 예측했고 실제로 생존 x
- FP(False Positive): 긍정예측을 실패 즉, 승객이 생존했다고 예측했지만 생존하지 못함
- FN(False Negative): 부정예측을 실패 즉, 승객이 생존하지 못했다고 예측했지만 생존함

(2) 정확도(Precision)
- True로 예측한 분석대상 중에서 실제 값이 True 인 비율
- 정확도 = TP/(TP + FN)


(3) 재현율(Recall)
- 실제 값이 True인 분석 대상 중에서 True로 예측하여 적중한 비율
- 재현율 = TP/(TP + FN)

(4) F1지표 (가장 정확한 지표)
- 정확도와 재현율이 균등하게 반영될 수 있도록 정확도와 재현율의 조화 평균을 계산한 값
- F1 지표 = 2*(정확도 * 재현율)/(정확도 + 재현율)
- 모형의 예측력을 종합적으로 평가하는 지표
- 값이 높을수록 모형의 예측력이 좋다.


머신러닝에서 제일 많이 쓰는 것: 분류, 회귀, 군집
2.군집분석
- 데이터셋의 관측값이 갖고 있는 여러 속성을 문석하여 서로 비슷한 관측값끼리 클러스터(집단)로 묶는 알고리즘
- 어느 클러스터에도 속하지 못하는 관측값이 존재할 수 있음 
	-> 특이 데이터(이상값, 중복값)를 찾는데 사용
- 머신러닝의 비지도 학습으로 분류

- 분류 vs 군집
	- 관측값을 몇 개의 집단으로 나눈다는 면에서 분류와 군집은 비슷하다
	- 군집분석은 정답이 없는 상태에서 데이터 자체의 유사성만을 기준으로 묶는다

- 신용카드 부정 탐지, 구매 패턴 분석 등 특성을 그룹화하는데 사용된다.
- K-Means, DBSCAN 알고리즘 등이 군집 알고리즘에 해당한다.


K-Means 알고리즘
- 클러스터의 중심까지의 거리가 데이터 간의 유사성을 측정하는 기준이 되어 분석하는 알고리즘
- K개의 클러스터가 주어졌을 때 클러스터의 중심까지의 거리가 가장 가까운 클러스터로 해당 데이터를 할당
- 다른 클러스터끼리 서로 일정 거리 이상 떨어져야 한다(서로 구분하기 위해)
- K값이 클수록 모형의 정확도 올라가지만, 너무 커지면 집단이 너무 많아져 분석의 효과가 떨어진다.
